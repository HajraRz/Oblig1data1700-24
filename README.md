Kort beskrivelse av applikasjon (5-10 setninger):
Jeg begynte med å opprette sideoppsettet i HTML-filen hvor jeg 
la inn knappene og inputene samt alternativ til filmer
som kunne velges. 
For å få det til å fungere slik jeg ville
opprettet jeg funksjoner i js-filen, funksjon for å fylle
inn i skjema hvor jeg også har valideringsskjekk og feilmeldinger,
funksjon for å vise bestillingen når alt er fylt ut riktig,
funksjon for å tømme skjemaet så den er klar for neste bestilling
og funksjon for å slette bestillingene.
